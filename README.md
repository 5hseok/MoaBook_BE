# MoA Book
## 구름톤 기업 연계 프로젝트 
단순히 정보를 저장하는 것이 아닌 저장하여 모은 정보를 자신만의 것으로 다시 기록할 수 있도록 유도하는 서비스 

---

### BE Part
**손수민** : 편집기 기능, URL 메타 데이터 가져오기 구현 

**오현석** : 서비스 기획, 이메일 알림, 배치 서버, 서비스 배포 및 인프라 설계 

**정원준** : ERD, OAuth 2.0 & Spring Security, CRUD 구현
